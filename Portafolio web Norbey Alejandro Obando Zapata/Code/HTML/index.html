Tools

HTML5


HTML is the fundamental language for creating web pages. I use it to structure the content in a semantic way and shape the basis of any website.

CSS3


CSS is essential for styling and designing web pages. I use it to control the appearance of HTML elements and create attractive and responsive interfaces.

Sass


SASS is a CSS preprocessor that simplifies writing and maintaining styles. It allows me to organize and reuse styles efficiently.

Bootstrap


Bootstrap is a CSS and JavaScript design framework that makes it easy to create responsive and attractive websites. I use it to speed up frontend development and ensure cross-browser compatibility.

Figma


Figma is an online collaborative design tool. I use it to create design prototypes, user interfaces and collaborate with other designers and developers on design projects.

Javascript


JavaScript is a programming language that adds interactivity and dynamism to web pages. I use it to create advanced functionalities and interactive user experiences.

React js


React is a JavaScript library for building interactive and reactive user interfaces. I use it to create single page web applications (SPA) with reusable components.

Node js


Node.js is a JavaScript runtime environment that allows me to develop server-side applications. I use it in conjunction with Express.js to create robust web servers and APIs.

Express js


Express.js is a Node.js web application framework that simplifies the creation of APIs and web applications. I use it to create routes, handle HTTP requests and develop efficient backend applications.

Mysql


MySQL is a relational database management system (RDBMS) that I use to store, manage and retrieve data. I use it to create robust and efficient databases.

Git


Git is a version control system that I use to track changes to source code and collaborate on development projects.

Github


GitHub is a cloud hosting platform that facilitates collaboration and change tracking on open source projects.
